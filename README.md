# 💬 Cmess - Корпоративный Консольный Мессенджер

**Cmess** — это легковесный, быстрый и безопасный консольный мессенджер, написанный на C++ с использованием сокетов. Предназначен для обмена сообщениями в локальной корпоративной сети без использования сторонних серверов.

---

## Особенности (Features)
- **Клиент-серверная архитектура**: Стабильная обработка нескольких подключений одновременно.
- **Real-time общение**: Мгновенная доставка сообщений без задержек.
- **Чистый консольный UI**: Использование ANSI-последовательностей для красивого вывода (автоматическое удаление введенной строки для предотвращения дублирования текста).
- **Валидация ввода**: Защита от отправки пустых сообщений или сообщений, состоящих только из пробелов.
- **Минимальные зависимости**: Написан на чистом C++ (STL) и системных API для работы с сетью.

---

## 🛠 Технологии
* **Язык**: C++ (стандарт C++_17_)
* **Сеть**: Winsock2 (Client)/ Boost.Asio (Server)
* **Сборка**: CMake
